TarsusFramework - Next NodeJS FrameWork

This is a http and microservice framework using TypeScript Decorator(state3).

Use TypeScript

The TarsusFramework use TypeScript as the basic building language, and it also supports JavaScript.

Support Http And MircoService

This framework not only provides support for http services, but also supports microservices.

Multiple Languages Available

This framework provides a structure called. taro, which follows the protocol provided by the framework and can be compiled into. ts and. java files.

Faster Stringify

We compile .taro into a JSON schema during compilation and efficiently serialize the returned values before returning them to the gateway (using fast-json-stringify).
